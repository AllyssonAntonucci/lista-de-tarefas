# Lista de Tarefas
Projeto desenvolvido com o framework Angular.

## Funcionalidades
Os usuários podem:

- Adicionar novas tarefas;
- Visualizar tarefas;
- Atualizar tarefas criadas;
- Excluir tarefas;
- Marcar tarefas como concluídas ou pendentes;

## Tecnologias Utilizadas no Projeto

- Angular;
- Typescript;
- HTML;
- SCSS;
